# Daily Meal Plan

A simple web app for planning your daily meals.

## Getting Started

1. Install dependencies:
   ```zsh
   npm install
   ```
2. Start the development server:
   ```zsh
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `public/` — Static files (HTML, manifest, icons, service worker)
- `src/` — Source code (main app logic)
